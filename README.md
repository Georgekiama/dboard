## Getting Started

### Prerequisites

- Node.js 16+ installed
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:3000` to view the app.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This app is configured for easy deployment on Vercel.

## Configuration

Edit the N8N webhook URLs in `src/App.tsx` to connect to your automation workflows.
